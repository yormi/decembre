# "Crowding" reclassified — symptom of over-aging, not a cell-size fault

Decided 2026-06-07, building the salanova growing protocol (`../goal.md`,
Stage 2 sowing). The diagnosis (`../problem.md`) listed four compounding
nursery faults: **salted, cooked, crowded, over-aged**. Grilling the
Stage-2 cell-size decision collapsed crowded into over-aged.

## Decision

- **Keep 50-cell** (1.25 L). Do **not** bump to 32-cell / soil block.
- A 50-cell is correctly sized for a **4-week** plug. It only read as
  "crowded" because the plug was held to **5–8 weeks** — roots outgrew the
  cell *because of age*, not because the cell was too small from the start.
- Four faults → **three**: salted · cooked · over-aged. "Crowded" is a
  symptom of over-aged, not an independent cause.

## Consequences

- **Salt fix relocates entirely to Stage 4** (cut the marine feed + leach
  the trays). The small 1.25 L cell does **zero** salt-buffering — there is
  no media-volume dilution to lean on. Stage-4 leaching must therefore be
  aggressive and EC-pen monitored.
- Pull-age (Stage 5) becomes the lever that "crowding" pretended to be:
  pull at the 4-wk weight peak, before roots bind the cell.

## Rejected: bigger cell (32-cell / soil block)

- Would have added root volume + salt buffer, but at bench-space and
  handling cost — and it treats a symptom (root-bound cell) whose real
  cause is holding too long. Fixing pull-age gets the same root quality
  without the bench cost.
- Soil-block's one unique benefit (no plug-wall interface → cleaner
  root-out, a Stage-7 transplant-shock win) is noted but not worth the
  labor here; revisit only if Stage-7 root-out stays stuck after the
  peat/calcareous-interface fix.
