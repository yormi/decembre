## Why

Thin new growth. N is **both** supply-tight *and*
uptake-impaired — a feed-side ramp helps but under-delivers
until roots recover (P/pH, SG1). A fresh SME closes the
supply-vs-uptake gap.

### Interim root-side — Ocean fish hydrolysate, DRENCHED
- Lever: EZ-GRO Ocean 15-1-1 fish protein hydrolysate, applied
  as a **root drench / fertigation** (not foliar).
- Mechanism: (1) amino-acid/peptide N taken up intact by roots
  → lower-barrier N for an impaired root than nitrate
  (no reduction step); (2) biostimulant peptides drive
  **root growth** (+21 % root DW in trial) → more absorptive
  surface for all uptake-limited elements. Drench beats foliar
  because a spray can't grow roots. Rating 5/5, confidence
  ~80 %. Full mechanism: `../tomato/learnings/0002-interim-root-uptake-levers-during-ph-wait.md`.
- NOT a P source (15-1-1 ≈ 1 % P₂O₅) — element delivered is N;
  root-growth benefit is the bigger, cross-cutting win.
- **Budget the N:** 15 % N is a real load — fold the drench
  into the N plan so it doesn't stack with the sidedress ramp
  into a CE overshoot. **Drench rate TBD — set inside the N
  budget.**
- Cert: allowed, already on hand + in use.
- Status: proposed; **next step = set drench rate vs sidedress.**
