# nutrition/lettuce/strategy — salanova yield-recovery strategy

Farm-level strategy for the salanova yield gap (60 → 200 kg/bed).
Rumelt kernel: **diagnosis → guiding policy → coherent action.**

- `problem.md` — the diagnosis: the crux (**the seedling is crap** —
  salted, cooked, over-aged — and a hot, salty field
  guarantees it never recovers; sodium + heat the mechanism), what's
  critical vs noise, supply-vs-uptake, two clocks, bad-strategy traps.
- `goal.md` — guiding policy (**outrun the heat**) + the 9-stage
  growing protocol (mix → crop-out), stage-organized. Stages 1–5 +
  8-climate decided; 6 (bed), 7 (transplant), 8-Ca/density, 9
  (targets) still open. References `../domain.md` for mechanism/setpoints.
- `fertigation-recipe.md` — field-feed coherent action: the
  tissue-led recipe revision (before vs corrected + reason +
  soil-reserve runway). Why K was cut, Mg moved to foliar, Cu kept,
  Mo + citric dropped. Shows depletion ≠ the constraint; lockout is.

- `learnings/` — strategy forks (crowding=over-aging → 3″-pot vehicle).
- `protocol/` — field operating procedures: `soil-ph-ec-test.md`
  (1:1 slurry EC/pH bands, salt-vs-pH gate) + `salt-leach.md`
  (plain-water reclamation, drainage half of the sulphur program;
  leach under the ECe 1.3 guardrail before each pass).
- `diagnostic-photos.html` — standalone photo viewer with a batch
  selector: each evidence photo with its ranked diagnoses (confiance
  ★/5 each) and the visual symptoms that justify each one. One data
  file per photo session, each registering a batch into
  `window.LETTUCE_DIAGNOSIS_BATCHES`:
  `diagnostic-data-2026-06-06.js` (8 nursery photos) +
  `diagnostic-data-2026-06-12.js` (33 field/tray photos). Generated by
  the lettuce-photo-diagnosis workflow (one agent per image). Open
  directly; references the gitignored photos by relative path. To add a
  session: convert photos into a dated `../../lettuce/doc/diagnostic-<date>/`,
  re-run the workflow, write a new `diagnostic-data-<date>.js`, add a
  `<script>` for it in the viewer.

Evidence base: `../../lettuce/doc/diagnostic-2026-06-06/`.

**Reach for these when:** deciding any salanova nutrition / climate /
propagation move; checking a proposed action against the diagnosis
before spending on it; onboarding why heat/CO₂/light + more fish are
the wrong levers.
