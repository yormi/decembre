# Seedling domain bands hold true root-zone EC/pH, not the meter reading

2026-06-13: Chose to store the *true root-zone* band in `../domain.md` and let each protocol convert it to a meter reading, rather than storing the method-reading directly (the alternative, considered then rejected). This keeps the biological setpoint method-free and puts the +0.10 tap-water EC correction in one place (the pour-through protocol), so an agent can't double-correct. Supersedes the "all domain rows are method-tagged" framing in `setpoint-band-provenance.md` for the seedling pH/EC rows; bed rows (Pulse, 1:1) stay method-tagged until paired offsets exist — and paired offsets only exist at fixed moisture, since the in-soil Pulse tracks moisture not salt (`pulse-in-soil-tracks-moisture-not-salt.md`).
