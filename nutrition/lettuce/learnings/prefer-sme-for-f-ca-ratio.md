# Read f_Ca from SME, not Mehlich-3

2026-06-13. The cation-antagonism ratio `f_Ca` (pathway 2.2 of outer-margin burn) is read from **SME** (saturated media extract = soil solution), not Mehlich-3. Uptake competition is a solution-phase event, and Na — the binding antagonist here — sits disproportionately in solution (weakly exchange-held), so the M3 reserve-pool view understates it.

This looks like it breaks the standing rule *"SME ranges don't apply to soil; lead deficiency reads with M3 + tissue"* — but it doesn't. That rule governs **absolute sufficiency** ("is there enough Ca?"). `f_Ca` is a **ratio within one SME sample** — self-referential, never compared to a soilless chart — so the SME-not-soil-calibrated objection doesn't apply.

Mehlich-3 still owns its own job: base-saturation / Na% of CEC (sodicity, the chronic-accumulation picture). Tissue Ca confirms the plant is actually short. SME, M3, and tissue answer three different questions; f_Ca is the SME one.
