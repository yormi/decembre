# Tomato yield-recovery subgoals — 2026-05-30

Priority frame for the 2026 tomato decline. Tomato yield ranks
above lettuce yield; this is the level *under* that priority —
target states (subgoals), root-cause first.

Evidence base: `tomato-t5-tissue-analysis-2026-05-23.md` (cert 4
tissue panel) + pour-through CE reading 2026-05-30 (Bluelab
~1.5 vs 3.0 target → root zone dilute, total supply low).

## Diagnosis — one crisis, three faces

The reported triad — thin new growth, flower abortion, botrytis
in a *dry* house — is not three problems. It's one pH-7.4
lockout crisis manifesting three ways (cert 4 tissue, cert 3-4
causal):

| Symptom | Nutrition root cause(s) |
|---|---|
| Thin new growth | N déficient (2.27% vs 3.10 floor) **and** Ca déficient (cell-wall structure; Ca is immobile → new growth starves first) |
| Flower abortion | P −90% (0.23 vs 0.50) — lockout starves the ATP/energy + root mass flowering needs. **NOT boron** (tissue B 57 ppm sufficient) |
| Botrytis (dry house) | Ca below resistance band (3.46 vs 4.5–5.5) + P+Ca wound-healing failure at old scars; rot colonises weak/aborted tissue — secondary, not the driver |

P starvation is upstream: it limits root mass, which throttles
N *and* Ca uptake. Hence root-cause-first ordering.

|---|---|---|---|---|---|---|---|

**Caveat the ranking hides:** #1 is first by causation but
slowest to move — start it now as a parallel clock. #2 is what
visibly reverses thin growth this crop. Rank = 1→2→3; calendar =
all start now, finish on different clocks.

## Non-nutrition levers (outside specialist lane — flag, don't own)

High-leverage moves nutrition can't make. Surfaced for Guillaume
/ team-leader, not specialist-owned:

- **Flower abortion → pollination: RESOLVED.** Bumblebee hives
  are running. Pollination is handled, so the abortion is a
  nutrition/energy problem (P, SG1), not a fruit-set gap.
- **Flower abortion → heat / light.** High night temperature +
  low light abort flowers (pollen sterility, assimilate
  shortage). Sun-only tomatoes — an environmental cap.
- **Botrytis → biofungicide.** Living wound/canopy biocontrol:
  *Bacillus subtilis* (Serenade) or *Gliocladium catenulatum*
  (Prestop). **Cert: typically organic-listed — verify the
  specific product's CAN/CGSB-32.311 / Ecocert listing before
  use, do not assume.**
- **Botrytis → airflow.** HAF-fan air movement dries the
  micro-layer around scars (distinct from humidity, already
  optimised).
- **Botrytis → pruning discipline.** De-leaf on a dry sunny
  morning so wounds suberize before evening; infections cluster
  at weeks-old scars.

## Monitoring — tissue is the decision-grade instrument

Decision: **tissue tests only.** Sap meter, SPAD, and Brix
rejected — too noisy/sensitive for decision-grade reads.

- All three KPIs (P, N, Ca) read off the periodic tissue panel.
- **Soil pH** (Bluelab pen, already on hand) is the one cheap
  leading gauge kept — it tracks SG1's driver directly and is a
  soil reading, not a noisy plant proxy.
- **Soil EC via direct pen insertion** is a coarse total-salt
  guardrail only, NOT a feed-quality or N gauge: target
  ~2.0–3.0 mS/cm at field-capacity moisture (read at consistent
  timing post-irrigation; moisture-dependent, cert 2). Below
  ~1.5 = underfed (current state), above ~3.5 = salt-stress risk.
  It does NOT track N — N comes from feather-meal mineralization,
  not the salt load — so tissue remains the N instrument.
