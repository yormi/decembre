# What the sulphur pH lever actually addresses (tomato), per the T5 tissue panel

**Date:** 2026-05-30
**Status:** accepted

The handoff framed the sulphur programme as fixing four goals at
once — drop foliar sprays, fix tissue deficiencies, restore vigor,
botrytis resilience. Cross-checked against the T5 tissue panel
(`nutrition/tomato/doc/tomato-t5-tissue-analysis-2026-05-23.md`),
the pH lever reaches each goal only through specific elements, with
different directness. Recorded so the programme isn't over-scoped.

## Element → does pH-drawdown help, and how

| Tissue deficit | Real cause | pH lever |
|---|---|---|
| **P −90 %** | pH-lockout (SME P 1.1 vs 5–50; Actisol P stuck at pH 7.4) | **Direct.** The one clean case. |
| **Micros (Fe/Mn/Zn)** | Tissue reads suff/élevé ONLY because the foliar spray props them up; root uptake is pH-suppressed (model applies Fe ×0.15 reductase lockout) | **Direct — lifts root uptake → lets the foliar micro spray be retired.** |
| **Ca déficient** | NOT pH-bound (Ca fully available at SME 6.99). Gap is uptake-side: P-starved root mass + K/Mg cation antagonism | **Indirect only** (pH→P→root mass→Ca). The antagonism half is NOT pH — and its fertigation lever is already spent (see below). |
| **N −26 %** | Feather-meal under-applied (supply gap) | **None** — sidedress ramp lever. |

## Correction logged

An earlier read in this session said "sulphur only addresses P."
That was too narrow. Tissue-sufficient micros mask a pH-suppressed
root pathway the foliar spray is compensating for, so relieving pH
genuinely enables dropping the foliar micro spray (a stated goal).
The multi-goal case largely holds — but each non-P payoff is
indirect (Ca) or works by enabling a different change (micros),
and all are slow/multi-cycle and bounded by a small P bank
(SME 1.1 ppm; no Ecocert foliar-P rescue exists).

## What pH does NOT carry

- **N** — sidedress Farine ramp.
- **Ca antagonism half** — NOT pH. The fertigation K₂SO₄/MgSO₄ trim
  for this was **already done 2026-05-28** (T5 MgSO₄ 1000 g, K₂SO₄
  5500 g) for cation antagonism. **Lever now essentially spent:** K
  is at tissue floor (can't cut further without starving K) and
  K₂SO₄ is the only organic-allowed soluble K (muriate prohibited),
  so its S is structural baggage — the +300 % S over-supply is
  largely unavoidable while feeding K through fertigation. Sulphur
  ADDS S, so it remains the wrong tool here regardless. Remaining Ca
  levers: foliar Ca (in since 2026-05-28) + the slow pH→P→roots path.
