# Microthiol / Kumulus DF as the soil-S source — agronomic risks (cert aside)

**Date:** 2026-05-30
**Status:** accepted — usable for soil pH correction with cadence + dose guardrails; one true unknown (adjuvant load)

Microthiol Disperss / Kumulus DF = ~80% elemental sulphur ground to
~9 µm, formulated as a wettable/dispersible-granule **fungicide**.
In soil the end-state chemistry is identical to any elemental-S
amendment (S⁰ → H₂SO₄ via *Thiobacillus*), so the deltas vs plain
micronized powder come only from particle size + formulation, not
from a different reaction. Cert question (off-label soil use of an
OMRI fungicide) set aside at Guillaume's request — this records the
soil/plant risk only.

## Risks specific to this product

| Risk | Cert | Note + mitigation |
|---|---|---|
| **Sharper acid pulse → overshoot** | 4 | 9 µm wettable oxidises ~3–5× faster than 200-mesh micronized → steeper local pH drop. Buffered while carbonate remains; bites fast once it thins. **Retest pH every 2 wk (not 4); smaller per-pass increments.** |
| **Fungicidal hit to rhizosphere fungi** | 3 | Contact fungicide at *soil-amendment mass* (kg/bed, not g/L spray) → real antifungal contact in the root zone. Two-sided: transient loss of beneficial/mycorrhizal fungi (tomato + lettuce low-dependence on fertigated calcareous soil → small) but incidental suppression of soil-borne fungal pathogens. **S-oxidisers are safe — *Thiobacillus* is a bacterium, not a fungus.** |
| **Formulation adjuvant load** | 2 | DF/Disperss carry dispersants + wetting agents (lignosulfonate-type) sized for foliar dilution, not bulk soil loading. At soil masses the adjuvant dose concentrates; surfactant in soil → possible local water-repellency/wettability oddities. Low risk, **genuinely unmeasured for this use — the one true unknown.** |
| **H₂S root burn on wet bed** | 4 | Same elemental-S anaerobic risk as any source, **amplified** by fast oxidation + full dispersion. Never dose a waterlogged bed; keep aerobic. |
| **Gypsum EC pulse** | 4 | Identical to any S source (CaSO₄ self-limits ~2.0–2.2 dS/m). Not Microthiol-specific. |

## Verdict

No soil/plant showstopper if dosed at the **same elemental-S caps**
(standing 2.5 / tilled 5.4 kg-S per 54.7 m² bed, `derivation.md`) with
pH retest at **2-wk** cadence. The two genuine deltas vs plain
micronized: faster overshoot (manage by cadence) + a transient
rhizosphere-fungal perturbation (mostly benign here, possibly
net-positive against pathogens). Adjuvant load = only true unknown.

Operator-safety note (not soil/plant): respirable fine-S dust on
handling; foliar SO₂-burn caution if dust drifts onto warm canopy.
