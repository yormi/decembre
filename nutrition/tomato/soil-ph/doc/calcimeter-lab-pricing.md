# Calcimeter test — lab pricing (research)

**Status:** cert 2 — estimates, NOT quoted figures. Confirm by phone
before ordering. ±50% band on every dollar figure.

**Why this exists:** the `soil-ph/model/spec.md` entry is blocked on a
free-carbonate (% CaCO₃, Bernard calcimeter) reading from the tomato
bed soil. The sulphur dose rule needs the **%**, not a fizz yes/no
(`S = 0.32 kg/kg CaCO₃`; a strong HCl fizz spans 2–15% CaCO₃ → ~7×
swing in sulphur demand). This doc scopes what that reading costs.

## Reading the table

- **Standalone ≈ 2–3× the add-on** everywhere — labs charge a minimum
  sample-handling fee whether you order one test or ten.
- **Lab spread < uncertainty.** Don't pick by price; pick by which lab
  already routes the routine panel (cheapest add-on, same sample).
- **Marginal cost to unblock the spec** = the add-on column (~$10–20)
  if a routine panel is already going out.
- **Cheapest full read:** Agro Enviro Lab, ~$40–63.

## Berger caveat (load-bearing)

Berger is a **growing-media maker**, not a soil-fertility lab. Their
analytical service is built around their own substrates (peat/coir,
SME readouts):

- Likely does NOT run a Mehlich-3 field/bed-soil panel.
- Calcimetry on mineral bed soil is likely off-menu — free carbonate
  is not a substrate-QC parameter.
- An SME-based read does not apply to mineral soil beds (SME ranges
  are hydroponic/soilless — see the `sme-spec-ranges` rule).

Right call for nursery-substrate QC; wrong tool for the bed-soil
free-carbonate number.
